## 设计原则

1. **简洁** - 界面简洁清晰，减少视觉干扰
2. **一致** - 遵循 Ant Design 设计规范
3. **高效** - 优化操作流程，提升工作效率

## 样式架构

本项目使用三层样式架构：

1. **Ant Design ConfigProvider（主题层）** - 主题 Token
2. **Tailwind CSS（布局层）** - 布局样式
3. **CSS 覆盖文件（细节层）** - 特殊样式覆盖
