## 项目概述

项目管理助手是一个轻量级 AI 驱动的桌面应用，帮助用户管理项目文件、与 AI 协作分析。

### 核心功能

1. **项目管理** - 创建、查看、管理项目
2. **文件管理** - 上传、分类、分析项目文件
3. **AI 对话** - 与 AI 助手对话，获取项目分析
4. **智能分类** - AI 自动分类文件到对应阶段

### 目标用户

- 需要管理多个项目文件的个人开发者
- 需要 AI 辅助分析项目文档的团队
